# oracle2

Prediction and forecasting engine (v2).

**Status:** Early stage — scaffolded, building, tests passing.

## What it does

Takes structured input and produces forecasts — probabilities, rankings,
or time-series predictions. Designed as a library that downstream agents
call into when they need reasoned estimates rather than guesses.

## Building

```sh
cargo build
cargo test
```
